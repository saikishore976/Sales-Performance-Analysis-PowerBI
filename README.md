## Tools Used
Power BI, SQL, Excel

## KPIs
- Total Revenue
- Month-over-Month Growth
- Average Order Value
- Profit by Category

## Key Insights
- Identified underperforming categories
- Highlighted top revenue regions
- Provided actionable recommendations
